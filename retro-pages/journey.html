임팩트 여정맵 회고

팀이 만들어낸 임팩트의 여정을 시각적으로 표현하며 돌아보는 회고입니다.

PARTICIPANTS
3명
TYPE
Visual Journey

설명

팀의 임팩트를 여정 지도(Journey Map)로 시각화하는 회고 방법입니다. 단순히 과거를 돌아보는 것을 넘어, 팀이 만들어낸 변화와 영향력의 흐름을 시각적으로 표현합니다. 히어로즈 저니에서 발견한 이야기들을 지도 위에 배치하며 팀의 임팩트를 확인하고, 앞으로 나아갈 방향을 함께 그려봅니다.

적합한 상황

  • • 팀의 임팩트를 시각화하고 싶을 때
  • • 복잡한 여정을 한눈에 정리하고 싶을 때
  • • 시각적 사고를 선호하는 팀
🐰 POPO'S NOTE

히어로즈 저니에서 발견한 이야기들을 시각화했어요. 여정 지도를 그리면서 팀의 임팩트가 어떻게 연결되고 확장되는지 명확하게 보였습니다.
