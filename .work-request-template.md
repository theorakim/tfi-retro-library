## 📝 추가 정보

### 회고 상세 페이지 작성 가이드

**설명 섹션** (2-3 문단):
- 1문단: 이 회고가 무엇인지, 어떤 구조로 되어있는지
- 2문단: 핵심 특징이나 세부 내용
- (선택) 3문단: 추가 정보

**적합한 상황** (3-4개):
- 구체적인 상황 설명
- "~할 때", "~이 필요할 때" 형식

**포포의 메모**:
- 이 회고의 특징
- 어떤 팀/상황에 적합한지
- 실제 사용 경험 기반 인사이트

**카테고리 종류**:
- FRAMEWORK: 구조화된 프레임워크 (4L, KPT 등)
- CUSTOM: 팀 맞춤형 (따로 또 같이)
- STRUCTURED: 체계적 접근 (사업별 회고)
- VISUAL: 시각적 접근 (임팩트 여정맵)
- CHRONOLOGICAL: 시간 순서 (타임라인)
- TRANSITION: 전환점 중심 (마무리와 시작)
- NARRATIVE: 스토리텔링 (히어로즈 저니)
- PLANNING: 계획/점검 (시동 점검)
